# Task-4-Sales-Financial-Dashboard
An interactive Sales Financial Dashboard built in Power BI using the Sample Superstore dataset. The dashboard provides insights into sales, profit, orders, quantity, category-wise performance, regional sales, and yearly trends.

## Project Overview
This project is a Power BI Sales Financial Dashboard created using the Sample Superstore dataset.

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Quantity

## Visualizations
- Sales by Category
- Sales by Region
- Profit by Sub-Category
- Sales by Year
- Region and Category Slicers

## Tools Used
- Microsoft Power BI
- Sample Superstore Dataset
